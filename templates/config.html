Gemini/Google Cloud TTS Voice

Warning! Setting you API key here could incurr charges if you exceed your free tokens. Use with care.

Step 1: Set Up Your Project & API

  1. Go to the Google Cloud Console and sign in.
  2. Select or create a new project using the dropdown menu at the top of the page.
  3. In the main search bar, type Cloud Text-to-Speech API and select it.
  4. Click the blue ENABLE button if the API is not already active.

Step 2: Create a Service Account & Download Key

  1. In the search bar, search for and navigate to the Credentials page.
  2. Click + CREATE CREDENTIALS and select Service account.
  3. Give your service account a name (e.g., my-tts-app-key), then click CREATE AND CONTINUE, and finally click DONE.
  4. On the Credentials page, click the email address of the service account you just created.
  5. Go to the KEYS tab, click ADD KEY, and choose Create new key.
  6. Ensure JSON is selected as the key type and click CREATE. A JSON file will be downloaded to your computer.

Step 3: Secure and Use Your Key

Important: Treat this downloaded JSON file like a password. Do not share it publicly, as it provides access to your Google Cloud account.

You can now paste the path to this file here.

Choosing a model

Choosing a model that is compatible with your device is important for the best performance.

Make sure the model doesn't exceed the memory limit of your device.

If you have a powerful GPU choose Kokoro. If in doubt, choose Piper. If Piper is too slow, try Gemini.
