Gemini/Google Cloud TTS

Warning: Setting your API key here could incur charges if you exceed your free tier. Use with care.
Show detailed setup instructions

Step 1: Set Up Your Project & API

  1. Go to the Google Cloud Console and sign in.
  2. Select or create a new project.
  3. Search for and enable the Cloud Text-to-Speech API.

Step 2: Create a Service Account & Download Key

  1. Navigate to the Credentials page.
  2. Click + CREATE CREDENTIALS and select Service account.
  3. Follow the prompts to create the account.
  4. Go to the KEYS tab for your new service account, click ADD KEY, and create a new JSON key. A file will be downloaded.

Step 3: Secure and Use Your Key

Treat this JSON file like a password. Paste the file path into the input box above.

General Settings & Info

Chunk Size

Smaller chunks generate faster but may have less natural pronunciation. Larger chunks provide better context but may be slower.

Additionally, Coqui does not work well with big chunks, clipping the text when it exceeds the limit. If you are planning to use Coqui, keep the Chunk below 50.

Choosing a Model

Choosing a model compatible with your device is important for performance.

  • Kokoro: Best for powerful GPUs.
  • Piper: A good default choice for most systems.
  • Gemini: A good cloud-based option if Piper is too slow.
